## Summary

Factory interface for creating [SimpleMessageBusFileProcessor](/simplemessagebus/api-reference/CloudNimble/SimpleMessageBus/Dispatch/Triggers/SimpleMessageBusFileProcessor) instances. This factory pattern allows
            different FileProcessors to be used for different job functions.
